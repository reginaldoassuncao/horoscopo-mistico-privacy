🔮 1. Informações que Coletamos

📱 Dados Locais (AsyncStorage)

  • ⭐ Signos favoritos selecionados pelo usuário
  • 🔄 Último signo visualizado para melhor experiência
  • ⚙️ Preferências e configurações do aplicativo
  • 📊 Estatísticas de uso local (não compartilhadas)

📺 Google AdMob (Anúncios)

  • 🔑 Identificador de publicidade do dispositivo (IDFA/GAID)
  • 👆 Informações sobre interações com anúncios
  • 📱 Dados técnicos do dispositivo (modelo, sistema operacional)
  • 🌍 Localização aproximada para anúncios relevantes
  • 🕐 Horários de uso do aplicativo

🎯 2. Como Usamos as Informações

  • 💫 Dados Locais: Personalizar sua experiência astrológica
  • ⭐ Favoritos: Salvar seus signos preferidos entre sessões
  • 💰 AdMob: Exibir anúncios relevantes e manter o app gratuito
  • 📈 Analytics: Melhorar a qualidade do conteúdo místico

🔒 3. Compartilhamento de Dados

🚫 NÃO vendemos ou compartilhamos seus dados pessoais.
🛡️ Apenas o Google AdMob processa dados conforme sua própria política de privacidade.
💻 Todos os dados místicos ficam salvos localmente no seu dispositivo.

🎮 4. Controle do Usuário

  • 🗑️ Limpar dados: Use o botão "Limpar Favoritos" no app
  • 📱 Desinstalar: Remove completamente todos os dados locais
  • 🔧 Controlar anúncios: Configurações → Google → Anúncios
  • 🚫 Opt-out de anúncios: Configurações do dispositivo

🛡️ 5. Segurança dos Dados

  • 🔐 Dados criptografados pelo sistema operacional
  • 🚫 Não coletamos informações pessoais sensíveis
  • 🔑 Não requeremos cadastro, login ou senhas
  • 📱 Armazenamento local seguro (AsyncStorage)
